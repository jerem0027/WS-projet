# Webservice - Train journey booking service

## Train Searching

Filters:

- Departure & Arrival
- outbound datetime [& return datetime]
- number tickets
- Class : First, Business, Eco

## Train Booking

Require :

- Train ID
- Travel class
- Ticket type (flexible, not flexible)

## SOAP

- Authentification
- Searching
- Reservation

## Rest

This WS provide response to SOAP requests
